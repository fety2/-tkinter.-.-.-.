# -tkinter.-.-.-.
графічним інтерфейсом за допомогою модулю tkinter. Головне вікно повинно містити меню з вибором потрібної функції. При виборі повинно створюватись нове вікно з полем для введення даних, для яких буде виконуватись дія. Після підтвердження введення програма виводить результати та пропонує повторити дію для нових даних або вийти у головне меню.
